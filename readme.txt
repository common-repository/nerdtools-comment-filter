== Description ==

Created to work alongside the NerdTools Bad Bots Spam Reporter & Spam Defender plugins, this plugin will read all comments posted to detect common spam elements and proceed to mark the comment as "spam" automatically if a match is found.

More information can be found at http://www.nerdtools.co.uk/badbots/

== Installation ==

1. Upload `nerdtools_comment_filter.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Head to 'Settings' then 'NT Comment Filter' and check 'Enable Comment Filtering' and press 'Save Changes'
4. That's it!
